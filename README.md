# MineSweeper

数设大作业

## Sram 使用情况

Sram 大小为 $2^{20}\times 32$

$00000000000000000000\sim 00000000000000001111(0\sim 1023)$ 用于存储格子信息，从上往下从左往右编号。

六边形的边长 $n=5$ 先行写死。

### 格子信息

- 0：是(1)否(0)有雷
- 1，2：无（00），翻开（01），插旗（10）,
- 总格子数 $2\sum_{i=n}^{2n-1}i-(2n-1)=(3n-1)n-2n+1=3n^2-3n+1$

- 前一半：$\sum_{i=n}^{2n-1}i=\frac12n(3n-1)$
